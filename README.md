# WhippyTermPlugin_IODriverHTTP
A WhippyTerm IO Driver for HTTP connections

This is a plugin for WhippyTerm (https://whippyterm.com) that adds support for opening HTTP (1.1) GET request
and transfers the body of the request as data.  The plugin handles all of the HTTP protocol.

# Installing
Download the plugin, start WhippyTerm and Plugins->Install Plugin menu item.  Select the .wtp file and click open.
